PROBLEM: Cannot login
SOLUTION: 
- Check database connection in config.php
- Verify database was imported correctly
- Default credentials: admin/admin123

PROBLEM: SMS not sending
SOLUTION:
- Verify API token in config.php
- Check mobile number format (947XXXXXXXX)
- Ensure cURL is enabled on server
- Contact Richmo support

PROBLEM: Page shows errors
SOLUTION:
- Check all files are uploaded
- Verify database connection
- Check PHP version (need 7.4+)
- Check error_log for details

PROBLEM: Can't delete customer
SOLUTION:
- Customers with appointments cannot be deleted
- Delete appointments first, then customer
